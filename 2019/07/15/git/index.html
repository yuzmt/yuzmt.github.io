git的常见使用命令

Posted on 2019-07-15 | 次阅读
git的基本命令

  • git clone

    将远程服务器的git库中的代码克隆岛本地的当前目录

将远程服务器的git库中的代码克隆岛本地的当前目录

  • git pull

​ git pull origin develop

​ 当本地已经存在相应的git库时，可以使用 git pull 从服务器同步最新的库到本地

  • git checkout develop

​ 切换到 develop 分支

​ git checkout -b zmt 创建分支 zmt, 并切换到 zmt 分支

​ git checkout -b zmt origin/develop 创建 zmt 分支并拉取 develop 分支

  • git add . 提交当前修改到本地缓存

    git add (文件名称) 提交单一文件

  • git commit -m '...' 提交本地缓存修改的内容到本地git库

  • git merge zmt 将 zmt 分支的内容合并到当前分支

    在做merge和push的时候，会出现代码冲突（conflict）的问题，git会自动合并可以合并的冲突，无法合并的冲突，需要手工进行冲突合并

  • git push将本地库代码推到服务器的git库

其他常用的git命令

  • git status 可以查看当前分支的状态，如代码是否被修改、新增的代码等

  • git branch `查看git库分支

    git branch -A `查看远程库有哪些分支

  • git log 查看git库日志

  • git reset 回退到git库的历史提交commit
