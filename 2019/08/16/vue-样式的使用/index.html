使用 style 样式

  • 直接在元素上通过 :style 的形式, 书写样式

    值得注意的是, 如果属性里面包含 - ,例如 font-size, font-family 等 , 就需要为其加上单引号

  • 当 style 里面的样式过多, 就显得可读性比较差, 我们也可以将样式定义到 date 中, 然后引用到 :style 即可
  • 在 :style 中通过数组, 引用多个 data 上的样式对象
