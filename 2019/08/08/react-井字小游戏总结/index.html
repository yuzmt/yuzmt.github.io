不可变性在 react 中非常重要

有两种改变数据的方式:

  • 直接修改变量的值

  • 新数据替换旧数据

有几点我们是需要注意的:

  • state 对于每个组件来说都是私有的,所以不能直接通过子组件 Square 来更新 父组件的 Board 中的 state
  • 在 React 中，有一个命名规范，通常会将代表事件的监听 prop 命名为 on[Event]，将处理事件的监听方法命名为 handle[Event] 这样的格式。
