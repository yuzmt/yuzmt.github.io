Chrome 浏览器的跨域设置

有时会遇到在启动一个项目之后，控制台会报下面的这个错误：

此时, 我们需要做的事情就是: 配置 Chrome 的属性

1. 右击 Chrome 的快捷方式, 选择属性;
2. 如图, 在 目标 的那个输入框的最后面加上下面的语句, 加上语句之后, 我们还需要在 C 盘的目录下新建 MyChromeDevUserData 这个文件夹 (不一定非要叫这个名, 但是必须保证文件夹的名称和目标输入框里的名称是一致)

语句: --user-data-dir=C:\MyChromeDevUserData

注意: –user 与前面是有空格的

3. 此时, 再重新浏览器, 重启项目即可
